# AI Tutor

Interactive AI tutoring app (Python + Gradio) with streaming responses and adjustable explanation levels.

- Features: streaming responses, 5-tier explanation slider, local LM Studio integration, error handling/fallbacks.
- How to run:
  1. Create a `.env` file with `OPENAI_API_KEY` and `OPENAI_BASE_URL`.
  2. Install deps: `pip install -r requirements.txt`
  3. Run notebook / start app.

Tech: Python, OpenAI SDK, Gradio, LM Studio, Jupyter
